> Le script de duplication automatique le plus rapide et le plus fiable pour voler des brainrots !

## 📋 Description

**Auto Dup Vol** est un script Roblox avancé qui permet de **dupliquer automatiquement** les animaux volés dans les jeux de type "Brainrot". 

### ✨ Fonctionnalités

- ⚡ **Ultra Rapide** - Duplication en seulement 0.15 secondes
- 🤖 **Auto Dup** - Duplique automatiquement dès que vous êtes proche d'un animal
- 🎯 **Détection intelligente** - Repère automatiquement les animaux à voler
- 📦 **Gestion des slots** - Libère automatiquement les slots 1 et 2 pour duper à l'infini
- 🎨 **Interface déplaçable** - Positionnez la fenêtre où vous voulez
- ⌨️ **Raccourci clavier** - Activez/Désactivez avec la touche `V`
- 🔐 **Protégé par Key** - Sécurisé avec une clé d'accès
- 📊 **Compteur en direct** - Affiche le nombre de duplications effectuées

## 🎮 Comment utiliser

1. **Exécutez le script** avec votre exécuteur préféré
2. **Entrez la key** : `Retrouver la sur le discord !`
3. **Activez l'Auto Dup** (bouton ou touche `V`)
4. **Approchez-vous** d'un animal à voler (le prompt "Steal" doit apparaître)
5. **Regardez** la magie opérer ! 🎩✨

## ⌨️ Commandes

| Touche / Action | Description |
|----------------|-------------|
| `V` | Activer / Désactiver l'Auto Dup |
| Glisser la barre du haut | Déplacer la fenêtre |
| Clic sur "DÉMARRER" | Activer l'Auto Dup |
| Clic sur "ARRÊTER" | Désactiver l'Auto Dup |
